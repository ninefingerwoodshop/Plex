How it works: Select a user, then click shows/movies to hide them. This adds a hide-username label in Plex.

Step 1: Make hidden content disappear from a user's Plex view:
1. Open Plex Settings > Manage Library Access (or Users & Sharing)
2. Click the user you want to configure
3. Under Restrictions, find Exclude items with labels
4. Add the label hide-username (e.g., hide-gibbens)
5. Save. That user will no longer see any content with that label.

Step 2 (optional): Add "Hide from..." to the Plex right-click menu:
1. Install Tampermonkey browser extension
2. Click the Tampermonkey icon > "Create a new script"
3. Paste the contents from /filter/userscript
4. Save. Now right-click any show/movie in Plex to see "Hide from..." options.
